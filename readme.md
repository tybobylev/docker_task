Простенькое веб-приложение (python/django) авиакасса(по-сути, табло авиарейсов) + субд постгрес. Подключаться нужно браузером по localhost:8000.
Через докер-компоус билдятся и запускаются два контейнера, образы которых созданы на основе регистра "registry.astralinux.ru".
Файлы БД идут в подпапке с образом астры с пострегресом.
